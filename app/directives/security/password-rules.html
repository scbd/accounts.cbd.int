Invalid characters
The password cannot contain following character(s):
  • A space
  • A tab
  • A carriage return
  • A new line
  • A vertical tab
  • A form feed
Password length
The password must be at least 10 characters long.
Password complexity
The password must match at least 2 of the following conditions:
  • Contains a lowercase character [a-z]
  • Contains an uppercase character [A-Z]
  • Contains a number [0-9]
  • Contains a special character [!@#$%^&*()_+^<>?/~|[]-\]
